# Digitale Ketenkaart Gemeente Huizen

Dit project brengt de publieke digitale footprint rond gemeente Huizen in kaart. Het onderzoekt niet alleen directe gemeentelijke domeinen, maar ook domeinen van uitvoeringsorganisaties, verbonden partijen, gemeenschappelijke regelingen en leveranciersplatformen die onderdeel lijken van publieke dienstverlening.

Het doel is transparantie: welke digitale loketten gebruiken inwoners, wie beheert deze, welke technische basismaatregelen zijn zichtbaar, en welke soevereiniteitsvragen ontstaan rond hosting, cloud, CDN's en externe scripts?

De scans zijn niet-invasief en maken alleen gebruik van publiek beschikbare informatie.
